# ResumeRank — ATS Resume Checker

A simple web app that checks how well a resume matches a job description.

## Features
- Upload resume and job description (PDF or text)
- Get ATS score (0–100%)
- See matched and missing keywords

## Tech Used
- Flask
- spaCy  
- pypdf  

## How to Run

```bash
python -m venv env
env\Scripts\activate
pip install -r requirements.txt
python app.py
